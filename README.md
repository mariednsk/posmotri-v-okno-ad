# Посмотри в окно

Приложение для просмотра видео из окон городов России. Фильтрация по городу и времени суток, подгрузка видео, переключение роликов.

Автор: mariednsk

## Стек
HTML, CSS, Vanilla JS

## Запуск
Открыть index.html или через Live Server
